<task type="auto">
  <name>Task 2: Create citation formatting service</name>
  <files>backend/citation_service.py</files>
  <action>
Create CitationService class:

**format_citation(source_type, source_data, style):**
- PAPER: Extract from Paper model (title, authors, year, venue)
- CLAIM: Extract from Claim + Paper
- MANUAL: Use provided citation_data

**Styles:**
- APA: "{Author}. ({Year}). {Title}. {Venue}."
- MLA: "{Author}. "{Title}." {Venue}, {Year}."
- Chicago: "{Author}. {Year}. "{Title}." {Venue}."

**format_bibliography(citations, style):**
- Sort by style requirements (APA: alphabetical, MLA: alphabetical, Chicago: by type)
- Return formatted bibliography string

Use existing Paper model fields. Return formatted strings.
  </action>
  <verify>python -c "from backend.citation_service import CitationService; print('OK')"</verify>
  <done>CitationService formats citations in APA, MLA, Chicago styles</done>
</task>

<task type="auto">

  <action>
Add to memory_api.py (document citations section):

**GET /api/documents/{id}/citations**
- List all citations for document
- Include source_type, source_id, citation_data

**POST /api/documents/{id}/citations**
- Create citation (from memory or manual)
- Body: {source_type, source_id?, citation_data, position}
- Returns created citation

**PUT /api/documents/{id}/citations/{citation_id}**
- Update citation_data
- Returns updated citation

**DELETE /api/documents/citations/{citation_id}**
- Delete citation
- Returns 204

**GET /api/documents/{id}/bibliography**
- Generate bibliography from all citations
- Query param: style (apa/mla/chicago)
- Returns formatted bibliography string

Follow async patterns from memory_api.
