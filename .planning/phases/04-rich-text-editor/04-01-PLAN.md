<task type="auto">
  <name>Task 1: Create Document and DocumentVersion models</name>
  <files>backend/database/models.py</files>
  <action>
Add Document and DocumentVersion models to database/models.py:

**Document model:**
- id: String(36), primary_key, default=generate_uuid
- project_id: String(36), ForeignKey("projects.id", ondelete="CASCADE"), nullable=False
- title: String(255), nullable=False
- content: JSONB, default=dict (TipTap document JSON)
- content_hash: String(64), nullable=True (SHA-256 hash for change detection)
- citation_style: SQLEnum(CitationStyle), default=CitationStyle.APA
- created_at: DateTime(timezone=True), default=utc_now
- updated_at: DateTime(timezone=True), default=utc_now, onupdate=utc_now
- created_by: String(36), ForeignKey("users.id"), nullable=True

**DocumentVersion model:**
- id: String(36), primary_key, default=generate_uuid
- document_id: String(36), ForeignKey("documents.id", ondelete="CASCADE"), nullable=False
- content: JSONB, nullable=False (TipTap document snapshot)
- change_description: Text, nullable=True
- created_at: DateTime(timezone=True), default=utc_now
- created_by: String(36), ForeignKey("users.id"), nullable=True
- parent_version_id: String(36), ForeignKey("document_versions.id"), nullable=True (for branching)

**CitationStyle enum:** APA, MLA, CHICAGO

Add indexes:
- Document.project_id (for listing project documents)
- Document.updated_at (for sorting recent documents)
- DocumentVersion.document_id (for version queries)

Follow existing model patterns from Claim, Finding models (async patterns, timezone-aware timestamps).

<task type="auto">
  <name>Task 3: Create Document API endpoints</name>
  <files>backend/document_api.py</files>
  <action>
Create document_api.py with FastAPI router (prefix="/api", tags=["documents"]):

**Endpoints:**

1. GET /api/projects/{project_id}/documents
   - List all documents for project (sorted by updated_at DESC)
   - Returns: [{id, title, citation_style, created_at, updated_at}]

2. POST /api/projects/{project_id}/documents
   - Create new document with title
   - Body: {title: string, citation_style?: enum}
   - Returns: full document object with empty content

3. GET /api/documents/{document_id}
   - Get single document with content
   - Verify project ownership
   - Returns: {id, title, content, citation_style, created_at, updated_at}

4. PUT /api/documents/{document_id}
   - Update document content and/or title
   - Body: {content?: JSONB, title?: string, citation_style?: enum}
   - Auto-create DocumentVersion on content change (detect via content_hash)
   - Returns: updated document

5. DELETE /api/documents/{document_id}
   - Delete document (CASCADE deletes versions)
   - Verify project ownership
   - Returns: 204 No Content

Use Pydantic models for request/response (DocumentRequest, DocumentResponse).
Follow async/await patterns from memory_api.py.
Use get_async_session() dependency injection.
  </action>
