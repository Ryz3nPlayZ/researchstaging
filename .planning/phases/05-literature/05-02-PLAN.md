  <action>
Add new method to MemoryService class in backend/memory_service.py:

```python
async def extract_claims_from_paper(
    self,
    project_id: str,
    paper_id: str,
    pdf_text: str,
    paper_metadata: Dict[str, Any],
    max_claims: int = 20
) -> List[Claim]:
    """
    Extract claims from a research paper using LLM.

    Args:
        project_id: Project to associate claims with
        paper_id: ID of the paper being processed
        pdf_text: Full text extracted from PDF
        paper_metadata: Paper metadata (title, authors, year, etc.)
        max_claims: Maximum number of claims to extract

    Returns:
        List of extracted Claim objects
    """
```

Implementation steps:
1. Construct LLM prompt for claim extraction:
   - Ask for 5-20 key claims/findings from the paper
   - Request claim type (assertion, fact, finding, hypothesis, method)
   - Request confidence score (0.0-1.0)
   - Request relevant section/quote from paper

2. Call LLM service with prompt and PDF text
   - Use structured output if model supports it (JSON mode)
   - Fallback to text parsing if needed

3. Parse LLM response into structured claims
   - Extract claim text, claim type, confidence, supporting quote
   - Validate claim text is not empty
   - Filter by confidence threshold (0.5 minimum)

4. Create Claim objects using existing create_claim method
   - source_type=ClaimSourceType.PAPER
   - source_id=paper_id
   - claim_data includes: {section, quote, page_number, authors}
   - extracted_by="system"

5. Return list of created Claim objects

Error handling:
- Catch LLM API failures
- Log parsing errors
- Return partial results if some claims fail
- Raise ClaimCreationError if complete failure

Add prompt template to backend/prompts/extract_claims.txt:
- Instructions for claim extraction
- Output format specification (JSON)
- Examples of good claims

     if (!paper.pdf_url) {
       toast.error("No PDF available for this paper");
       return;
     }

     setProcessingPaperIds(prev => new Set([...prev, paper.external_id]));

     try {
       const result = await memoryApi.extractClaims(projectId, {
         paper_id: paper.external_id,
         pdf_url: paper.pdf_url,
         paper_metadata: paper,
         max_claims: 20
       });

       setExtractedClaimCounts(prev => ({
         ...prev,
         [paper.external_id]: result.data.length
       }));

       toast.success(`Extracted ${result.data.length} claims from paper`);
     } catch (error) {
       toast.error(`Failed to extract claims: ${error.message}`);
     } finally {
       setProcessingPaperIds(prev => {
         const newSet = new Set(prev);
         newSet.delete(paper.external_id);
         return newSet;
       });
     }
   };
