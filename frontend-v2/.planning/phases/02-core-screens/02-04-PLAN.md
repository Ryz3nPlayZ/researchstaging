  <name>Task 1: Create useTaskGraph hook</name>
  <files>frontend-v2/src/hooks/useTaskGraph.ts</files>
  <action>
    Create frontend-v2/src/hooks/useTaskGraph.ts:

    **Purpose:** Fetch and transform task data for ReactFlow graph visualization

    **Parameters:**
    - projectId: string

    **State:**
    - nodes: TaskGraphNode[]
    - edges: TaskGraphEdge[]
    - isLoading: boolean
    - error: Error | null

    **Functionality:**
    1. **Fetch project tasks:**
       - Call projectsService.getProject(projectId)
       - Extract tasks array

    2. **Transform tasks to graph format:**
       - For each task, create a TaskGraphNode:
         - id: task.id
         - type: 'task'
         - position: { x, y } (calculated or from backend)
         - data: { task object, status, name, type }

    3. **Create edges from dependencies:**
       - For each task dependency:
         - Create edge from dependency task to dependent task
         - id: `${dependency_id}->${task_id}`
         - source: dependency_id
         - target: task_id
         - type: 'dependency'

    4. **Layout calculation (simple version):**
       - If positions not provided, calculate simple layout
       - Use helper function from graphUtils
       - Grid layout or DAG layout (simplified for now)

    5. **Return:** { nodes, edges, isLoading, error }

    **Error Handling:**
    - Handle fetch errors
    - Return empty graph on error

    This hook provides graph-ready data for ReactFlow component.
  </action>
  <verify>useTaskGraph hook transforms tasks to nodes and edges</verify>
  <done>Task graph hook for data fetching and transformation</done>

    Create frontend-v2/src/utils/graphUtils.ts:

    **Purpose:** Helper functions for graph styling and layout

    **Functions:**

    1. **getTaskNodeColor(status: TaskState): string**
       - Map task status to color
       - pending: gray (#9ca3af)
       - running: blue (#3b82f6)
       - completed: green (#10b981)
       - failed: red (#ef4444)
       - blocked: orange (#f59e0b)
       - Use design token colors

    2. **getTaskNodeSize(type: TaskType): { width: number, height: number }**
       - Different sizes based on task type
       - search: { width: 200, height: 80 }
       - summary: { width: 200, height: 80 }
       - synthesis: { width: 220, height: 100 }
       - Default: { width: 180, height: 60 }

    3. **calculateLayout(tasks: Task[]): GraphLayout**
       - Simple grid or layered layout
       - Group by type or phase
       - Calculate x, y positions
       - Returns: Map<taskId, {x, y}>

    4. **formatLabel(text: string, maxLength: number): string**
       - Truncate long text
       - Add ellipsis if truncated
       - For task names in nodes

    5. **getEdgeType(dependencyType: string): string**
       - Map dependency type to edge style
       - smoothstep, straight, or default
       - Different styles for different dependency types

    **Implementation:**
    - Use TypeScript types from types/task.ts
    - Export all utility functions
    - Keep functions pure and testable

    These utilities support graph visualization and keep graph logic organized.
  </action>
