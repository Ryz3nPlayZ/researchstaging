<task type="auto">
  <name>Task 1: Create AI text assistance endpoints</name>
  <files>backend/document_api.py</files>
  <action>
Add to document_api.py (AI assistance section):

**POST /api/documents/{id}/ai/rewrite**
- Body: {selection: string, tone: string}
- Tones: formal, casual, concise, elaborate
- Call llm_service.generate() with prompt:
  "Rewrite this text in a {tone} tone. Keep the meaning but change the style.\n\nText: {selection}"
- Return: {rewritten: string}

**POST /api/documents/{id}/ai/grammar**
- Body: {text: string}
- Call llm_service.generate() with prompt:
  "Check this text for grammar, spelling, and style issues. Return corrected text and list of suggestions.\n\nText: {text}"
- Return: {corrected: string, suggestions: [{original, correction, explanation}]}

**Integration:**
- Import llm_service from backend.llm_service
- Use llm_service.generate() with system prompts
- Handle errors gracefully

Use existing LLM service patterns.
  </action>
