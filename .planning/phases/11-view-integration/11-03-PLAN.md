<task type="auto">
  <name>Task 2: Initialize TipTap editor in EditorView</name>
  <files>frontend3/pages/EditorView.tsx</files>
  <action>
Initialize TipTap editor and replace contentEditable:

**1. Add imports:**
```typescript
import { useEditor, EditorContent } from '@tiptap/react';
import StarterKit from '@tiptap/starter-kit';
import Link from '@tiptap/extension-link';
import Placeholder from '@tiptap/extension-placeholder';
import Underline from '@tiptap/extension-underline';
```

**2. Initialize editor with useEditor hook:**
```typescript
const editor = useEditor({
  extensions: [
    StarterKit.configure({
      heading: {
        levels: [1, 2, 3],
      },
    }),
    Link.configure({
      openOnClick: false,
    }),
    Underline,
    Placeholder.configure({
      placeholder: 'Start writing your document...',
    }),
  ],
  content: '<p>Start writing your document...</p>',
  editorProps: {
    attributes: {
      class: 'prose prose-sm sm:prose-base lg:prose-lg xl:prose-xl max-w-none focus:outline-none min-h-[500px] px-8 py-6',
    },
  },
});
```

**3. Find and replace contentEditable div:**
- Locate the contentEditable div in EditorView (search for "contentEditable")
- Replace the entire div with: `<EditorContent editor={editor} />`
- Remove any contentEditable attributes

**4. Add Tailwind prose styles:**
Add the prose classes to the editor content (already in editorProps above):
```typescript
className="prose prose-sm sm:prose-base lg:prose-lg xl:prose-xl max-w-none focus:outline-none min-h-[500px] px-8 py-6"
```

**Preserve existing UI:**
- Keep the floating toolbar (will wire in task 3)
- Keep the chat sidebar (do not modify)
- Keep the overall layout (editor left, chat right)
- Only replace the contentEditable area with TipTap

Do NOT modify the chat sidebar or toolbar yet - that's in task 3.
