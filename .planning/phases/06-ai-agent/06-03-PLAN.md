2. Add propose_plan() method to AgentRouter:
   ```python
   async def propose_plan(
       self,
       query: str,
       context: dict
   ) -> ProposedPlan:
       """Generate a plan for complex multi-step actions."""

       # Detect if query requires complex action
       if self._is_simple_query(query):
           return None  # No plan needed

       # Use LLM to generate plan
       prompt = f"""
       User request: {query}

       Available contexts:
       - Document: {bool(context.get('document'))}
       - Literature: {len(context.get('claims', []))} claims
       - Memory: {len(context.get('findings', []))} findings

       Generate a step-by-step plan to accomplish this request.
       Return JSON with steps, each with description and action_type.
       """

       response = await self.llm.generate_response(
           prompt=prompt,
           response_format="json"
       )

       return ProposedPlan.parse_raw(response)
   ```

3. Add _is_simple_query() helper:
   - Simple queries: "What's X?", "Tell me about Y", "Summarize Z"
   - Complex queries: "Analyze all papers and extract claims", "Compare X and Y across all documents"
   - Heuristic: Query length > 50 chars OR contains action verbs ("analyze", "compare", "extract", "generate")

4. Add to chat_api.py:

   const handleApplySuggestion = (suggestion, range) => {
     if (editorRef.current) {
       const { view } = editorRef.current;
       const { state } = view;
       const { from, to } = range;

       // Replace text
       const tr = state.tr.replaceWith(
         state.doc.mapping.map(from),
