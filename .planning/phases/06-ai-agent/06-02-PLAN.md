  <action>
Create new file backend/agent_service.py:

1. AgentRouter class:
   - analyze_query(query: str) -> str: Returns agent type
   - route(query: str, context: dict) -> AgentResponse
   - Agent types: "document", "literature", "memory", "general"

2. Base Agent class:
   ```python
   class Agent(ABC):
       def __init__(self, llm_service: LLMService):
           self.llm = llm_service

       @abstractmethod
       async def handle(self, query: str, context: dict) -> str:
           pass

       @abstractmethod
       def can_handle(self, query: str) -> float:
           """Return confidence score 0.0-1.0"""
           pass
   ```

3. Specialized agents:

   DocumentAgent:
   - Handles queries about current document
   - Keywords: "this document", "current text", "section", "paragraph"
   - Loads document content from context
   - Provides document-aware assistance

   LiteratureAgent:
   - Handles queries about papers, research, literature
   - Keywords: "papers", "literature", "citations", "references"
   - Searches memory for relevant claims/findings
   - Can search literature if needed

   MemoryAgent:
   - Handles queries about stored information
   - Keywords: "what did we find", "claims", "analysis", "remember"
   - Searches claims and findings by relevance
   - Provides synthesis of stored information

   GeneralAgent:
   - Fallback for general questions
   - No context specialization
   - General research assistance

4. Confidence scoring:
   - Each agent implements can_handle() with keyword matching
   - Router selects highest confidence agent
   - GeneralAgent has confidence 0.1 (always available)

5. Response structure:
   ```python
   class AgentResponse:
       agent_type: str
       confidence: float
       response: str
       context_used: dict
   ```

Integration:
- Import LLMService from llm_service.py
- Import MemoryService from memory_service.py
- Use async/await throughout
- Log routing decisions for debugging

1. Create inject_context() function:
   ```python
   async def inject_context(
       project_id: str,
       document_id: Optional[str],
       query: str
   ) -> dict:
       """Gather relevant context for AI response."""

       context = {}

       # Document content
       if document_id:
           doc = await get_document(document_id)
           context['document'] = {
               'title': doc.title,
               'content': doc.content,  # TipTap JSON
               'citation_style': doc.citation_style
           }

       # Recent claims (last 20)
       claims = await memory_service.list_claims(
           project_id=project_id,
           limit=20,
           min_confidence=0.5
       )
       context['claims'] = [
           {
               'text': c.claim_text,
               'type': c.claim_type,
               'confidence': c.confidence
           }
           for c in claims
       ]

       # Recent findings
       findings = await memory_service.list_findings(
           project_id=project_id,
           limit=10
       )
       context['findings'] = [
           {
               'type': f.finding_type,
               'summary': f.summary
           }
           for f in findings
       ]

       # User preferences
       prefs = await memory_service.get_preferences(project_id)
       context['preferences'] = {
           'domains': prefs.domain_preferences,
           'topics': prefs.topic_keywords
       }

       return context
   ```

3. System prompt construction:
   ```python
   def build_system_prompt(context: dict) -> str:
       base = "You are a research assistant helping with academic work."

       if 'document' in context:
           base += f"\n\nCurrent document: {context['document']['title']}"
           base += f"\nCitation style: {context['document']['citation_style']}"

       if context['claims']:
           base += f"\n\nRecent claims from memory: {len(context['claims'])} items"

       if context['preferences']:
           base += f"\n\nUser interests: {context['preferences']['topics']}"

       return base
   ```
