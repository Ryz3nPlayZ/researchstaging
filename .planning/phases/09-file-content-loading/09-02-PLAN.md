  <action>
Add `markdown_to_tiptap(markdown_text: str) -> dict` function that converts Markdown to TipTap JSON.

**Implementation approach:** Use regex-based parsing (no external markdown library dependency for MVP).

**Parse rules:**
1. **Empty input**: Return `{type: "doc", content: [{type: "paragraph", content: []}]}`

2. **Split by blocks**: Split markdown by `\n\n` to get blocks

3. **Block type detection** (process each block):
   - **Headings**: Match `^(#{1,6})\s+(.+)$` → heading node with level 1-6
   - **Code blocks**: Match ```^```(\w*)\n([\s\S]+?)\n```$``` → codeBlock node with language
   - **Blockquotes**: Match `^>\s+(.+)$` → blockquote node
   - **Unordered lists**: Match `^[\-\*]\s+(.+)$` → bulletList with listItem
   - **Ordered lists**: Match `^\d+\.\s+(.+)$` → orderedList with listItem
   - **Default**: paragraph node

4. **Inline parsing** (within blocks):
   - **Bold**: `\*\*(.+?)\*\*` → text with bold mark
   - **Italic**: `\*(.+?)\*` or `_(.+?)_` → text with italic mark
   - **Code**: `` `(.+?)` `` → text with code mark
   - **Links**: `\[(.+?)\]\((.+?)\)` → text with link mark
   - **Plain text**: Everything else → plain text node

**TipTap node structure:**
```python
# Paragraph
{"type": "paragraph", "content": [{"type": "text", "text": "Hello"}]}

# Heading
{"type": "heading", "attrs": {"level": 2}, "content": [{"type": "text", "text": "Title"}]}

# Bold text
{"type": "text", "text": "bold", "marks": [{"type": "bold"}]}

# Italic text
{"type": "text", "text": "italic", "marks": [{"type": "italic"}]}

# Bullet list
{"type": "bulletList", "content": [
    {"type": "listItem", "content": [
        {"type": "paragraph", "content": [{"type": "text", "text": "Item"}]}
    ]}
]}

# Code block
{"type": "codeBlock", "attrs": {"language": "python"}, "content": [
    {"type": "text", "text": "print('hello')"}
]}
```

```python
class FileParseRequest(BaseModel):
    file_id: str
    project_id: str

@router.post("/files/parse/tiptap")
async def parse_file_to_tiptap(
    request: FileParseRequest,
    db: AsyncSession = Depends(get_db)
):
    """
    Parse file content to TipTap JSON format for document editor.

    Returns TipTap JSON structure ready for DocumentEditor initialization.
    """
    try:
        from file_service import read_file_content, markdown_to_tiptap, FileServiceError

        # Read file content
        content_data = await read_file_content(db, request.file_id, request.project_id)
        content = content_data["content"]

        # Get file to determine type
        file_record = await get_file(db, request.file_id)
        extension = file_record.name.split('.').pop().lower()

        # Parse based on file type
        if extension == 'md':
            tip tap_json = markdown_to_tiptap(content)
        elif extension == 'docx':
            # For now, return simple paragraph structure for DOCX
            # Will be handled in plan 09-03
            paragraphs = content.split('\n\n').filter(p => p.trim())
            tiptap_json = {
                "type": "doc",
                "content": paragraphs.length > 0
                    ? paragraphs.map(p => ({
                        "type": "paragraph",
                        "content": [{"type": "text", "text": p.trim()}]
                    }))
                    : [{"type": "paragraph", "content": []}]
            }
        else:
            raise HTTPException(
                status_code=400,
                detail="Unsupported file type for TipTap conversion"
            )

        return {
            "file_id": request.file_id,
            "tiptap_json": tiptap_json,
            "format": "tiptap"
        }
    except FileServiceError as e:
        raise _handle_file_service_error(e)
    except Exception as e:
        logger.error(f"Error parsing file to TipTap: {e}")
        raise HTTPException(status_code=500, detail="Failed to parse file content")
```
