<task type="auto">
  <name>Task 3: Add document API client functions</name>
  <files>frontend/src/lib/api.js</files>
  <action>
Add document API functions to api.js:

**saveDocument(documentId, content, title?):**
- PUT /api/documents/{id}
- Body: {content, title?}
- Returns updated document

**loadDocument(documentId):**
- GET /api/documents/{id}
- Returns document with content

**createDocument(projectId, title, citationStyle):**
- POST /api/projects/{id}/documents
- Body: {title, citation_style}
- Returns new document

**listDocuments(projectId):**
- GET /api/projects/{id}/documents
- Returns array of documents

**deleteDocument(documentId):**
- DELETE /api/documents/{id}
- Returns 204

Follow existing API client patterns (use axios, handle errors, return data).
  </action>
