<task type="auto">
  <name>Task 1: Create version history API endpoints</name>
  <files>backend/document_api.py</files>
  <action>
Add to document_api.py:

**GET /api/documents/{id}/versions**
- List all DocumentVersion records for document
- Order by created_at DESC
- Returns: [{id, change_description, created_at, created_by}]

**POST /api/documents/{id}/restore/{version_id}**
- Restore document to version content
- Create new DocumentVersion for pre-restore state (audit trail)
- Update Document.content with version content
- Returns: restored document

Follow async patterns.
  </action>
  <verify>grep -q "versions" backend/document_api.py</verify>
