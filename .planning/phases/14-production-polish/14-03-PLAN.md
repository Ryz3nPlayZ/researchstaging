    2. Open browser: http://localhost:3000
    3. Execute each test flow from 14-03-TEST-PLAN.md:

    **Flow 1: Create Project**
    - Click "New Project" button in Dashboard
    - Enter research goal, select output type, add audience
    - Verify project appears in dashboard

    **Flow 2: Upload Files**
    - Navigate to Files view
    - Click "Upload Files" or drag-drop a test file
    - Verify file appears in list with correct metadata

    **Flow 3: Write in Editor**
    - Navigate to Editor view (click "New Document")
    - Type text, use toolbar buttons (bold, italic, underline, link, quote, list)
    - Verify formatting applies correctly

    **Flow 4: Citations**
    - Click "Insert Citation" button
    - Search for a paper (e.g., "machine learning")
    - Select a result, verify citation inserts
    - Verify bibliography appears below editor

    **Flow 5: Literature Search**
    - Navigate to Library view
    - Enter search query (e.g., "climate change")
    - Verify results appear with paper details
    - Verify source badges (arXiv/Semantic Scholar)

    **Flow 6: Data Analysis**
    - Navigate to Analysis view
    - Enter Python code: `print("Hello, World!")`
    - Click Execute
    - Verify output appears

    **Flow 7: Export Documents**
    - In Editor, click Export dropdown
    - Select "Export as PDF" - verify download
    - Select "Export as DOCX" - verify download

    **Flow 8: AI Chat**
    - In Editor sidebar, select each agent type (Document, Literature, Memory, General)
    - Send a test message to each
    - Verify responses appear

    **Flow 9: WebSocket Status**
    - Check toolbar for connection status (Live/Connecting/Offline)
    - Should show "Live" when backend is running

    **Flow 10: Auto-save**
    - Type text in editor
    - Wait 4+ seconds
    - Verify status changes from "Saving..." to "Saved"

    For each flow: Mark as PASS or FAIL. Document any issues in 14-03-TEST-PLAN.md.
