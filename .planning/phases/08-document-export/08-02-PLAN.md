<task type="auto">
  <name>Task 1: Create ExportButton component with dropdown</name>
  <files>frontend/src/components/editor/ExportButton.jsx</files>
  <action>
    Create ExportButton component following established UI patterns:

    1. **Component structure:**
       - Use Shadcn DropdownMenu (consistent with citation picker, AI features)
       - Trigger button: "Export" with download icon
       - Menu items: "Export as PDF", "Export as DOCX"

    2. **Props:**
       - documentId: ID of document to export
       - projectId: ID of project (for ownership validation)
       - documentTitle: Title for filename (fallback: "document")
       - disabled: Disable export while exporting

    3. **Export state management:**
       - exporting: boolean (true during export)
       - exportFormat: 'pdf' | 'docx' | null
       - error: string | null

    4. **API integration:**
       - Use api.exportDocumentPdf() and api.exportDocumentDocx() from lib/api.js
       - Pass documentId, projectId as query params
       - Handle blob response (FileResponse from backend)

    5. **Download handling:**
       - Create blob URL from response data
       - Create temporary anchor element with download attribute
       - Trigger download programmatically
       - Clean up blob URL after download

    6. **Loading indicators:**
       - Show "Exporting..." text while export in progress
       - Disable menu items during export
       - Show toast notification on completion (success/error)

    7. **Error handling:**
       - 404: "Document not found"
       - 403: "You don't have permission to export this document"
       - 500/503: "Export failed. Please try again."
       - Network error: "Connection error. Check your internet."

    8. **Styling:**
       - Follow Tailwind patterns from existing editor components
       - Hover states, focus states for accessibility
       - Icon: DownloadIcon from lucide-react
  </action>
  <verify>
    - ExportButton component renders with dropdown menu
    - PDF and DOCX menu items trigger respective API calls
    - Loading states display correctly during export
    - Download files have correct filenames and extensions
  </verify>
  <done>
    ExportButton component provides export UI with dropdown, API integration, download handling, loading states, and error handling.
  </done>
</task>
