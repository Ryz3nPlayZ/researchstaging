Create CitationPicker.jsx:

**Props:** documentId, projectId, onInsert, onClose

**Features:**
- Tab 1: "From Memory" - Search papers/claims
  - Search input queries /api/memory/projects/{id}/papers
  - Results show: title, authors, year
  - Click to insert citation at cursor position

- Tab 2: "Manual Entry" - Form for manual citations
  - Fields: authors, title, year, venue, url
  - "Insert" button creates MANUAL citation

- Citation style selector: APA/MLA/Chicago (per-document or global)
- Preview of formatted citation before insert

**Integration:**
- Use TipTap's insertContent() to insert citation mark
- Store citation position in document (for bibliography generation)

Follow dialog component patterns.

<task type="auto">
  <name>Task 2: Create Bibliography component</name>
  <files>frontend/src/components/editor/Bibliography.jsx</files>
  <action>
Create Bibliography.jsx:

**Props:** documentId, style

**Features:**
- Fetch citations from /api/documents/{id}/citations
- Call /api/documents/{id}/bibliography?style={style}
- Display formatted bibliography (rich text)
- Auto-update when citations change
- "Copy to clipboard" button

**Display:**
- Show heading: "References" (APA/MLA) or "Bibliography" (Chicago)
- List each citation with hanging indent
- Numbered (APA) or alphabetical (MLA/Chicago) per style

**Trigger:**
- Auto-generate when document loads
- Manual "Refresh" button

Position at bottom of document (auto-inserted page break).
  </action>
