  <name>Task 1: Create chat API endpoints</name>
  <files>backend/chat_api.py</files>
  <action>
Create new file backend/chat_api.py with FastAPI router:

1. GET /api/chat/projects/{project_id}/history
   - Returns paginated message history for project
   - Response: {messages: [{id, role, content, timestamp, context}]}
   - Supports limit/offset query params

2. POST /api/chat/projects/{project_id}/send
   - Accepts: {message: string, context: {document_id, selection, analysis_id}}
   - Stores user message in database (chat_messages table)
   - Calls LLM service with context injection
   - Stores AI response in database
   - Returns: {user_message, ai_response}

3. Chat message model (add to database/models.py if not exists):
   ```python
   class ChatMessage(Base):
       __tablename__ = "chat_messages"
       id = Column(String, primary_key=True)
       project_id = Column(String, ForeignKey("projects.id"))
       role = Column(String)  # "user" or "assistant"
       content = Column(Text)
       context = Column(JSONB, nullable=True)  # Optional context snapshot
       created_at = Column(DateTime, default=datetime.utcnow)
   ```

4. Context injection function:
   - Load document content if document_id provided
   - Load analysis results if analysis_id provided
   - Load recent claims/findings from memory
   - Load user preferences
   - Build system prompt with context

Integration:
- Import LLMService from llm_service.py
- Use AsyncSession for database operations
- Return structured responses with message objects
